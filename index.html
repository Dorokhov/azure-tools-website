Free, cross-platform, open source tools for Microsoft Azure developers

    Features

  • Redis Viewer

    • View, create and delete keys
    • Search by key ('*' wildcard is supported)
    • Bulk keys delete
    • Create, update, delete string, set and hash data structures
    • Import connection settings
    • HTTP connection is only supported at the moment

  • Tables Viewer

    • Browse tables
    • Create, delete tables
    • View and query table entities

  • Blobs Viewer

    • View blobs and containers
    • Search blob by prefix
    • Preview blob as a picture
    • Preview blob as a text
