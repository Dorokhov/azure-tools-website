Azure Tools
Free, cross-platform, open source Redis viewer
Download for Windows

    Features

    • Multiple Redis servers support
    • Filter Redis keys
    • Redis string, set, hash and zset supported
    • Group in folders by separator (':' by default)

    Known Issues and Limitations

    • Redis v. >= 2.8 supported
    • Unable to edit a server settings
    • Unable to edit key separator (':' is default)

    Under the Hood

    • Electron
    • Angular 4
